What is an LLM?

LLM stands for Large Language Model. It is a type of artificial intelligence model trained on huge amounts of text data to understand and generate human language.

LLMs work by predicting the next token in a sentence based on patterns learned during training. They are trained using books, websites, articles, and many other text sources.

Some real-world examples of LLMs are GPT-4o by OpenAI, Claude 3.5 Sonnet by Anthropic, Gemini 1.5 Pro by Google, and DeepSeek R1.

What is a Token?

A token is a small chunk of text used by LLMs. A token is usually about 3-4 characters or around ¾ of a word in English.

Tokens are important because they are used to measure context windows and API pricing. The more tokens used, the higher the processing cost.

Example: "Hello, world!" may be split into tokens like: "Hello", ",", "world", "!"

English usually consumes fewer tokens compared to Azerbaijani because Azerbaijani words are often longer and contain more suffixes.
